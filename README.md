# GovTech Complaint / Case Management System

This project is a web-based system that allows citizens to submit complaints related to public services and enables government officers to manage and resolve those complaints.

## Features

- Citizen registration and login
- Submit complaints
- Upload evidence (images or documents)
- Track complaint status
- Officer dashboard to manage complaints
- Admin monitoring and reports

## Technologies Used

Frontend:
- HTML
- CSS
- React.js
- Vite

Backend:
- Node.js

Database:
- MongoDB
