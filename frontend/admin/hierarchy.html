How Automatic Escalation Works

If an L1 officer doesn't resolve a complaint within 5 minutes, it escalates to their L2 supervisor. If the L2 also misses the deadline, it escalates to the L3 Director. Assign supervisors below so escalation always reaches the right person. If no supervisor is set, the system picks the least-loaded officer of the next level.
